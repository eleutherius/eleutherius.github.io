1. Создайте файл less/style.less или sass/style.scss с таким CSS-кодом:

body { background: red; }

2. В консоли в папке проекта запустите команду одноразовой сборки npm run build.

3. Удостоверьтесь, что после компиляции в папке css появится файл style.css. После чего подключите этот файл на свои страницы. Их фон должен стать красным.

Если всё сработало, то сборка работает.

4. В консоли в папке проекта запустите команду npm start. После чего внесите изменения в файл препроцессора, например поменяйте фон страниц на синий, сохраните файл.

Если страница перезагрузилась и изменения на страницах видны, то отслеживание изменений работает.

5. Создайте структуру препроцессорных файлов:

    каждому БЭМ-блоку должен соответствовать отдельный препроцессорный файл,
    файлы БЭМ-блоков импортируются в файле less/style.less или sass/style.scss,
    вспомогательные файлы с переменными, примесями и так далее создавайте на своё усмотрение.

6. Начните вёрстку вашего проекта с помощью препроцессора:

    задайте фоновые цвета и фоновые изображения для блоков;
    параметры текста: шрифты, размеры шрифта, межстрочные расстояния, цвет текста;
    добавьте внутренние и внешние отступы и обводки для крупных сеточных блоков.

Мелкие декоративные элементы можно не верстать.
